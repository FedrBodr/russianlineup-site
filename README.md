# russianlineup-site

Лендинг бренда «Русский Лайнап» — электросёрф, тест-дни, кампы, комьюнити. Статический сайт (один `index.html`), деплой из git по push в `master`.

## Что внутри

- `index.html` — лендинг. Кнопки ведут в Telegram-бота (deep-link с UTM + склейка через ClientID Яндекс.Метрики).
- `favicon.svg`, `robots.txt`, `sitemap.xml`, `CNAME` — SEO/инфраструктура.
- Аналитика: Яндекс.Метрика (приоритет РФ/СНГ) + GA4 в `<head>`.

## Перед публикацией

Открой `index.html`, в блоке `window.SITE_CONFIG` подставь реальные значения и замени плейсхолдеры `[в скобках]`. Полный список — в `CONTENT_TODO.md`.

## Деплой

См. `DEPLOY.md` (push в `master` → автосборка).

## Связь с воронкой

Кнопки лендинга → бот с параметром `start=<источник>__<ClientID Метрики>`. Бот (этап 7) разбирает это и сшивает веб-сессию с действиями в Telegram — единая сквозная аналитика. Репозиторий бота: https://github.com/FedrBodr/ruslineup-bot
